# NTU EMBA AI Course v2

New React/Vite source for the NTU EMBA Generative AI course timeline.

Important preservation rule:

- The original Netlify site is not modified.
- The original GitHub repository `600725abc/ai_course_2024_2025` is not modified.

## Development

npm install

```

```
